## Media-completeness gate

The current typed inventory includes direct `media` fields for LLM Steering, HL7 AI Challenge, and AlphaQuant. Other featured projects can still link to detailed routes and repository artifacts, but this manifest does not claim that every featured project already has a distinct screenshot or video rendered in the V2.3 Lab.

Before staging acceptance, inspect each required project for:

- problem statement;
- what exists;
- source and repository health;
- status and limitations;
- at least one truthful artifact, diagram, output, API example, test result, screenshot, or demo link;
- media source, commit, license, alt text, caption, and public-safety decision.

Any missing media must remain a visible release gap; it must not be replaced with fabricated output.

- media and detail route when present.

## Non-destructive policy

This manifest authorizes no repository deletion, archival, privacy change, relicensing, or pin mutation. Portfolio classification is reversible editorial curation. Live GitHub changes require separate explicit scope.
