# Shailesh Dudala Portfolio

GitHub Pages portfolio for Shailesh Dudala, Senior Applied AI / Agentic AI Engineer.

Site URL: https://smgpulse007.github.io/

## Local Development

```bash
npm ci
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

The site deploys to GitHub Pages with `.github/workflows/deploy.yml` using GitHub Actions.

No `CNAME` file is included. Custom-domain work is intentionally deferred.
