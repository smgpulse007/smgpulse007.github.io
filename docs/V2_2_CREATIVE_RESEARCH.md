# Portfolio V2.2 creative research

## Territory

The design borrows principles, not identities, from scientific observatories, medical imaging, aerospace telemetry, network topology, genomic maps, signal processing, and mission-editorial graphics.

The useful qualities are:

- a field with scale and depth instead of a stack of cards;
- changing system state instead of decorative movement;
- labels that behave like annotations on an instrument;
- connections that express causality or lineage;
- calm, luminous contrast rather than neon cyberpunk;
- a reading experience that remains editorial and human.

## Central visual grammar

The site uses a living signal line and six custom stage marks: Signal, Prediction, Product, Context, Agent, Harness. Nodes expand from raw observation into increasingly explicit contracts, authority, and proof. Warm white carries the narrative, mineral blue represents evidence, electric citron marks active state, and coral is reserved for exception or human authority.

## Anti-patterns rejected

- Particle wallpaper: motion without meaning.
- Generic glass dashboard: fragments the story and weakens reading order.
- Terminal cosplay: confuses code aesthetics with system depth.
- Game HUD: raises visual noise and damages professional tone.
- Universal cards and chips: makes projects look equivalent and hides causality.

## Capability tiers

- Enhanced: SVG path drawing, pointer-aware node emphasis, scene-linked motion.
- Standard: complete SVG diagrams, keyboard controls, semantic HTML, and small browser scripts.
- Reduced/static: still compositions, full text equivalents, tables/lists, no essential animation.

No WebGL dependency was selected. The prototype comparison showed that the distinctive idea is the topology and transformation itself; SVG preserves it with lower cost and a stronger mirror/no-JavaScript fallback.
