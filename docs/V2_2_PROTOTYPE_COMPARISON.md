## What each concept proved

### Observatory

One field can make the career legible as a connected system without reducing it to six cards. It supports the broadest future information architecture: projects, papers, eras, and proof can all attach to the same topology.

### Agent operating system

The authority boundary is the most memorable treatment of current work. Context, model, tools, state, validation, exception routing, and human review read as different layers rather than an “AI agent” blob.

### Evolution engine

The sequence Signal → Prediction → Product → Context → Agent → Harness is an effective narrative rail and mobile index. It is less effective as the entire first fold because it underweights the present-day system.

## Selected hybrid

The final direction uses:

1. Observatory as the first-fold spatial grammar and overall world.
2. Evolution as the permanent narrative spine and section-transition system.
3. Agent operating system as the current-frontier interaction and the coral authority language.

Citron remains the active signal/proof color, mineral blue carries evidence and topology, warm white carries editorial meaning, and coral is used only for exception/reviewer authority.

## Discarded

- Observatory-only: too abstract without the explicit agent sequence.
- Agent-only: makes the career appear narrower than it is.
- Evolution-only: reads as a polished timeline before it reads as a distinctive operating system.
- WebGL layer: the SVG prototype already expresses the idea with better performance, accessibility, no-JavaScript resilience, and GitHub Pages parity.
