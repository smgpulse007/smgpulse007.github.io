- production agentic claims automation;
- bounded autonomous adjudication support;
- human-controlled consequential decisions;
- typed workflow state;
- deterministic validation;
- exception routing;
- reviewer authority;
- trace telemetry;
- synthetic reconstruction;
- measured or supported workstream.

Do not publish:

- employer-confidential system names;
- private APIs, prompts, policy text, claim data, or production screenshots;
- exact private model routing;
- an implication that a model self-approves consequential exceptions;
