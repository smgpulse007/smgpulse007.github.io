# Portfolio V2.1 project evidence

Recorded: 2026-07-13

## Flagship evidence decisions

### Claims intelligence

- Source boundary: current career record and a synthetic reconstruction authored in this repository.
- Public visual: page identities, an ambiguous service date, typed fields, a validation failure, reviewer reason, and evidence-page references.
- Outcome scope: approximately 90% lower handling effort and approximately 50% shorter time-to-payable in the measured workstream.
- Limitation: no employer data, production screenshot, proprietary rule, API, prompt, document, or model route is published.

### On-prem document intelligence

- Source boundary: current career record plus the adjacent public `ollama_poc` experiment.
- Public visual: an authored local-boundary diagram and synthetic cited reviewer route.
- Outcome scope: 7,000 cases cleared and approximately 90% lower review time in the measured workflow.
- Limitation: the public experiment is not the employer system and does not prove an OCR path that is absent from its source.

### Healthcare analytics platform

- Source boundary: approved career record only.
- Public visual: an authored Health Index composite, nine-program adoption field, and model-to-workflow product map.
- Outcome scope: $500K new revenue supported and approximately $3M client performance-based payouts supported by the broader program; approximately 50% faster deployment cycles.
- Limitation: no private dashboard, client data, clinical metric, or employer artifact is reproduced.

### LLM Steering

- Repository: `https://github.com/smgpulse007/llm-steering`
- Source commit: `8c1242839e1f8f74fd50f4b6bad37f71a2d83122`
- License: MIT.
- Checked-in evidence inspected: workbench UI capture, activation/hook diagrams, pre/post media, and JSON comparisons in `docs/showcase/`.
- Public visual: a static baseline-versus-steered comparison using checked-in output and the recorded experiment configuration.
- Limitation: output effects depend on model, prompt, layer, token scope, and coefficient. A comparison is not causal proof or a general safety guarantee.
